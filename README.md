# ProMessage

Modern, professional messaging interface built with a modular component architecture.

## Features
- **Professional Design**: Single-pane layout focused on the conversation.
- **Mobile Friendly**: Responsive design with smooth transitions and touch-optimized UI.
- **State-Driven**: UI updates automatically via a central store.
- **Component-Based**: Over 50 files ensuring separation of concerns.
- **Animations**: CSS-driven animations for message bubbles and view transitions.

## Architecture
- `state/`: Custom store for application state.
- `components/`: Atomic UI components.
- `services/`: Logic for data fetching and authentication.
- `styles/`: Modular CSS files for different app sections.
- `utils/`: Common helper functions.
